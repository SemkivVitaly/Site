# ERP система для типографии

Полнофункциональная мини-ERP система для автоматизации бизнес-процессов типографии.

## Технологический стек

- **Frontend**: React + TypeScript + Material-UI
- **Backend**: Node.js + Express + TypeScript
- **Database**: PostgreSQL + Prisma ORM
- **Real-time**: Socket.io
- **Authentication**: JWT + bcrypt
- **PWA**: Service Worker для оффлайн-режима

## Структура проекта

```
typography-erp/
├── frontend/          # React приложение
├── backend/           # Express API
├── shared/            # Общие типы и утилиты
└── database/          # Миграции и схемы
```

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Настройте переменные окружения (см. `.env.example`)
4. Запустите миграции БД:
   ```bash
   cd backend
   npx prisma migrate dev
   ```
5. Запустите проект:
   ```bash
   npm run dev
   ```

## Роли пользователей

- **ADMIN** (Начальник производства) - полный доступ
- **MANAGER** (Менеджер) - работа с заказами и клиентами
- **EMPLOYEE** (Сотрудник) - упрощенный мобильный интерфейс

## Основные модули

- Управление заказами (Канбан-доска)
- Управление производством (технологические карты, задачи)
- HR и учет смен (QR-коды, электронная проходная)
- Рабочее место сотрудника (мобильный интерфейс)
- Система уведомлений (WebSocket)
- Складской модуль
- Аналитика и КПД
